# My API - Gin Web 應用程式

基於 Gin 框架的 Go Web API 專案

## 快速開始

```bash
# 安裝依賴
go mod tidy

# 運行應用程式
go run main.go
```

應用程式將在 `http://localhost:8080` 啟動
